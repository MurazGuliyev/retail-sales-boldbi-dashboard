# Retail Sales Performance Dashboard – Bold BI / MS SQL Server

## Project Overview

This project is an interactive retail sales performance dashboard built with Bold BI and MS SQL Server. The dashboard was designed to monitor daily sales performance, branch-level results, product and category performance, payment distribution, customer count, sales returns, and period-based comparisons.

## Tools & Technologies

* Bold BI
* MS SQL Server
* T-SQL
* Dashboard Parameters
* KPI Cards
* Line Charts
* Bar Charts
* Donut Charts
* Gauge Widgets
* Data Visualization
* ERP / Retail Sales Data

## Key Dashboard Features

* Daily sales, return amount, net sales, customer count, and average basket KPI cards
* Daily sales target tracking with gauge visualization
* Hourly net sales analysis
* Branch-based sales and customer count comparison
* Top 10 product, category, department, and supplier analysis
* Payment type distribution analysis
* Card vs non-card customer sales comparison
* Cashier-based sales and return comparison
* Last 15 days, last 6 months, month-over-month, and year-over-year sales comparison
* Dynamic filters by branch, date, month, day, and hour

## Dashboard Pages

### 1. Main Dashboard

Shows daily sales KPIs, net sales, returns, customer count, average basket value, and daily sales target.

### 2. Charts & Analysis

Includes hourly sales, payment type distribution, customer card analysis, Top 10 products, categories, departments, suppliers, and cashier-based sales/return analysis.

### 3. Branch Comparison

Compares branches by net sales, customer count, hourly performance, daily sales, and returns.

### 4. Period Comparison

Analyzes sales trends for the last 15 days and last 6 months by branch.

### 5. Month-over-Month and Year-over-Year Comparison

Compares current period sales with previous month and previous year performance.
